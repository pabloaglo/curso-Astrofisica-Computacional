En este programa se recibe una cantidad N (ingresada por el usuario) de números enteros (también ingresa-
dos por este) en desorden y se usa el método Bubble Sort para ordenarlos de menor a mayor.
Se consultó el algoritmo del método y se implementó en C por medio de dos funciones.
Los números ingresados por el usuario se guardan en un archivo que posteriormente es leído y se guardan
estos en un arreglo declarado de dimensión N. Se recorre este arreglo con la función Burbuja y se orga-
nizan los números. Posteriormente, el arreglo organizado es guardado en un nuevo archivo.

Comandos utilizados:
	gcc Punto6.c -o Punto6.exe
	./Punto6.exe
	>>Se ingresa la cantidad de números
	>>Se ingresan los números
