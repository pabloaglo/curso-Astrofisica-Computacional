Este programa crea dos matrices distintas y realiza las operaciones básicas con estas, no sin antes com-
probar que se puedan realizar, con el uso de arreglos multidimensionales.

Las matrices son de dimensiones ingresadas por el usuario, con elementos enteros también ingresados por
este. Si se van a tomar matrices de dimensiones grandes se le pide al usuario que tome elementos de
menor tamaño para no obtener resultados basura producidos por problemas de memoria.
El programa cuenta con un menú para escoger qué operación se desea realizar con el uso de un ciclo do...
while, donde el usuario decide si continúa con el programa o desea terminarlo.

Comandos utilizados dentro de la misma carpeta:
	
	gcc Punto2.c -o Punto2.exe
	./Punto2.exe
	>>Se crean las matrices
	>>Se crean los elementos de las matrices
	>>Se escoge la operación deseada en un loop
	>>Se decide si continuar el loop o terminar el programa
